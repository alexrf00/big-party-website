# Big Party — Website

Marketing site for **Big Party**, an online western party brawl. Round up to 12 outlaws
for a bite-size showdown — quick matches, dirty tricks, and one dusty crown.

## Contents

| File | Description |
|------|-------------|
| `index.html` | The website. Loads fonts from Google Fonts; all images are local under `assets/`. |
| `assets/` | Images used by the site (hero art, pixel props). Swap these to update visuals. |
| `Big Party - Bright Daybreak (offline).html` | The same site as a single self-contained file — works fully offline. Just open it in a browser. |

## Running locally

It's a fully static site — no build step. Either:

- Open `index.html` directly in a browser, or
- Serve the folder, e.g. `python -m http.server` and visit `http://localhost:8000`.

## Deploying

Static hosting. `index.html` is at the repo root, so it deploys as-is on Vercel,
Netlify, GitHub Pages, or any static host — no configuration required.
