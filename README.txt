BIG PARTY — website (Bright Daybreak direction)
================================================

WHAT'S IN HERE
--------------
index.html ........................... The website. Host this, or open it in a
                                        browser. Loads fonts from Google Fonts
                                        (needs internet); all images are local.
assets/ .............................. Images used by index.html (hero art,
                                        pixel props). Swap these to update the site.
Big Party - Bright Daybreak (offline).html
                                        Same site as a single self-contained file.
                                        Works fully offline — no internet, no other
                                        files needed. Just double-click it.

QUICK START
-----------
• Just want to look at it?  Open the "(offline)" file.
• Putting it on the web?     Upload index.html + the assets/ folder together.

NOTES
-----
• Screenshot / trailer areas are polished placeholders ("GAMEPLAY SHOT — SOON").
  Send real screenshots and they can be dropped straight in.
• The newsletter form is a visual placeholder — it isn't wired to a mailing
  service yet. Point it at Mailchimp / ConvertKit / etc. when you're ready.
• Roadmap milestones are drafts (Playtests → Early Access → New Modes → 1.0).
• Links wired: Wishlist → Steam, plus Discord / Instagram / TikTok in the footer.
